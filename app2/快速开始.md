# ScoreKeeper 原型快速开始指南

## 🚀 如何查看原型

### 方法一：浏览器直接打开（推荐）

1. **打开主入口文件**
   ```
   双击打开: prototype/index.html
   ```
   这会在默认浏览器中打开原型展示页面

2. **查看效果**
   - 所有8个原型页面会以手机屏幕形式平铺展示
   - 每个页面都模拟了iPhone 16 Pro的真实尺寸
   - 可以滚动查看所有页面

### 方法二：单独查看各个页面

如果你想单独查看某个页面的详细效果：

```
prototype/home.html      - 首页
prototype/room.html      - 房间页面
prototype/score.html     - 计分页面
prototype/share.html     - 分享页面
prototype/stats.html     - 统计页面
prototype/history.html   - 历史记录
prototype/profile.html   - 个人中心
prototype/join.html      - 加入房间
```

### 方法三：本地服务器（可选）

如果需要测试完整的iframe功能：

```bash
# 使用Python快速启动本地服务器
cd prototype
python -m http.server 8080

# 然后在浏览器访问
http://localhost:8080/index.html
```

或使用VS Code的Live Server插件：
1. 安装Live Server扩展
2. 右键点击index.html
3. 选择"Open with Live Server"

## 📱 原型页面说明

### 1. 首页 (home.html)
**核心功能**：
- 创建新房间按钮（主要CTA）
- 扫码加入房间
- 快速入口网格（4个快捷功能）
- 进行中的对局列表
- 底部TabBar导航

**交互要点**：
- Logo有发光效果
- 按钮有按压反馈
- 活跃对局卡片可点击

---

### 2. 房间页面 (room.html)
**核心功能**：
- 房间信息卡片（名称、ID、人数、时长、状态）
- WebSocket连接状态指示灯（脉冲动画）
- 操作按钮组（分享、统计、结束）
- 玩家列表（头像、昵称、在线状态、分数）
- 实时排行榜（前3名特殊样式）
- 浮动加分按钮（右下角）

**设计亮点**：
- 房主有特殊徽章
- 分数正负用不同颜色（青色/红色）
- 第1名有金色冠军徽章
- 连接状态实时显示

---

### 3. 计分页面 (score.html)
**核心功能**：
- 当前操作者展示
- 接收者选择（多选网格）
- 分数显示区（超大字体）
- 快速分数按钮（+1/+5/+10/+50）
- 数字键盘（0-9+退格）
- 底部确认按钮

**交互要点**：
- 选中的接收者有高亮边框和发光效果
- 分数实时显示
- 数字键盘有按压反馈

---

### 4. 分享页面 (share.html)
**核心功能**：
- 房间信息展示
- 二维码展示区（白色背景+圆角）
- 房间ID大字体显示+复制按钮
- 分享方式网格（微信、朋友圈、保存图片）
- 使用说明卡片

**设计细节**：
- 二维码区域有阴影效果
- 房间ID使用等宽字体
- 分享图标用品牌色

---

### 5. 统计页面 (stats.html)
**核心功能**：
- 收支统计概览（2个大卡片）
- 筛选标签（全部、收入、支出、回合）
- 转移记录列表（时间、玩家、分数）
- 收入/支出不同颜色区分

**数据展示**：
- 总收入：+385分（青色）
- 总支出：-100分（红色）
- 详细流水记录
- 回合号标注

---

### 6. 历史记录 (history.html)
**核心功能**：
- 筛选标签（全部、已完成、已过期、时间）
- 按日期分组（今天、昨天、本周）
- 对局卡片（名称、状态、成绩、排名）
- 排名徽章（金/银/铜）

**信息展示**：
- 对局Emoji图标
- 状态徽章（已完成/已过期）
- 我的成绩（大字体）
- 参与人数、时长、排名

---

### 7. 个人中心 (profile.html)
**核心功能**：
- 用户信息卡片（头像、昵称、ID、数据）
- 功能菜单分组（8组）
- 账户、数据、设置、帮助等入口
- 版本信息

**菜单分类**：
- 账户相关：编辑资料、我的好友
- 数据与统计：数据统计、成就勋章
- 设置相关：消息通知、主题设置、通用设置
- 帮助与反馈：帮助中心、意见反馈、关于我们
- 退出登录：红色警告样式

---

### 8. 加入房间 (join.html)
**核心功能**：
- 标签切换（扫码/输入ID）
- 扫码界面（大图标+说明+按钮）
- 输入界面（房间ID+昵称）
- 最近访问列表
- 加入提示卡片

**交互逻辑**：
- 点击标签可切换界面
- 有简单的JavaScript实现

## 🎨 设计规范

### 颜色系统
```css
/* 主色调 */
主色: #22d3ee (霓虹青绿)
强调色: #ff48f9 (霓虹紫粉)

/* 背景色 */
深色背景: #0f172a
次级背景: #1e293b

/* 文字色 */
主文字: #ffffff
次要文字: #94a3b8
禁用文字: #64748b

/* 状态色 */
成功/收入: #22d3ee
错误/支出: #ef4444
警告: #f59e0b
```

### 尺寸规范
```css
/* iPhone 16 Pro */
屏幕尺寸: 393×852 px
状态栏: 44px
TabBar: 80px
圆角: 20px (卡片), 50px (手机外框)

/* 间距 */
页面边距: 20px
卡片间距: 12-15px
元素间距: 8-10px
```

### 字体规范
```css
/* 字号 */
标题: 22-24px (font-weight: 900)
正文: 14-16px (font-weight: 400-600)
小字: 12-13px (font-weight: 400)
大数字: 24-64px (font-weight: 900)

/* 字体 */
系统字体: -apple-system, BlinkMacSystemFont, 'Segoe UI'
等宽字体: 'Courier New', monospace (用于ID显示)
```

## 💡 开发建议

### 1. 直接使用
- 所有代码都是纯HTML+CSS，可以直接复制使用
- 已使用CDN引入Tailwind CSS和Font Awesome
- 可作为微信小程序的WXSS参考

### 2. 样式提取
如果要用于实际开发，建议：
1. 提取公共样式到单独的CSS文件
2. 将颜色、尺寸等定义为CSS变量
3. 组件化设计（如按钮、卡片等）

### 3. 响应式适配
当前原型固定为iPhone 16 Pro尺寸，实际开发时需要：
- 使用相对单位（rem、em）
- 添加媒体查询
- 考虑不同设备的安全区域

### 4. 性能优化
- 图片使用真实资源（当前使用占位符）
- 考虑懒加载
- 动画使用transform和opacity（硬件加速）
